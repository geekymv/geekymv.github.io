可以看到，mygit 目录下多出了.git 目录，这个子目录包含初始化的 Git 仓库所有必须的文件，
比如有我们在上一篇文章中介绍的关于git config –local配置方式中提到的config文件。

一般我们不会直接修改这个文件，而是通过Git 提供的命令去操作。
